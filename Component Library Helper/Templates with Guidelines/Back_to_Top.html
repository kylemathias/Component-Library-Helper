Gives users a shortcut back to the top of a long page.

 

Template Copy Block

 

Component name: Back to Top

Guidelines:

 

Label (required, not author editable, translates)

Back to Top

 

 

Use When

  • A long page that contains mostly prose or accordions would benefit from a jump link back to the top of the page. 

Don't use when

  • The final component on the page has in-component footnotes (such as a product comparison table). 
  • The page has page-level footnotes. These automatically add a "back to top" function that is redundant with this component. 

Editorial

Label (required, not author editable, translates)

Visual Style

Placement

  • Use in the main content region, below Prose but above any parting CTAs, such as Next Steps in Offset Cards. See example below. 
  • Use only once per page.
