Use When

  • Grouping multiple sets of peer navigation links that go to related pages. This is particularly relevant for groups of products or solutions. 

Don’t Use When

  • Tiles can't be meaningfully grouped. Use Image with Tiles instead. 

 

Editorial

Pill (optional) Image area only

  • Link to the relevant solution hub (only) using this distinctive eyebrow 'tag'.

Headline (required)

  • Text field.
  • Character limit is 45.

Body  (required)

  • Rich text field. Bullets, inline links, and paragraphs allowed. 
  • Character limit is 100.

CTA Link (optional)

  • Text link only. Label is required.
  • Character limit is 30.

Image (required)

  • Background images must not hinder the readability of copy.
  • See Visual Style for image composition and sizing guidelines.

Tabs

Tab Labels (required)

  • Minimum of 2 tabs.
  • Text field.
  • Character limit is 15.

Tiles

Headline (required)

  • Text field.
  • Character limit is 30.

Body  (required)

  • Rich text field. Bullets, inline links, and paragraphs allowed. 
  • Character limit is 60.

CTA Link (optional)

  • Text link only. Label is optional. If no label provided an arrow will still appear.
  • Character limit is 15.
  • Entire tile is interactive if linked.
