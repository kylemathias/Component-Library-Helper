Use When

Showcasing at least 3-6 of NetApp's customers or partners. 

Don't use when: 

  • Displaying 1-3 customers. Consider quote band/quote band tabbed 

Editorial

Per tile

  • Logo image (required) 
  • Alt text (required) Always contains organization name. 
  • URL (optional).
    • Default is not to link.
    • Link all tiles or link none.
    • Link internally (netapp.com, preferred) or externally (e.g. to partner site). 

View All tile (optional, preview below)

  • Special, optional tile. Use when you have too many customers/parters to show. Always use with a full row of 5 linked logos. 
  • Always the last item in the list, appearing right-most (i.e. last). 
  • Has a special style that isn't themed. 
  • Text is always "View all" (+ translation).

Visual Style

Placement

  • Use in the main content region.
  • Can be used multiple times on a page.

Layout

  • Does not mirror. 
  • How many logos to use:
    • Use 3-6 tiles on a single row. This is the preferred default. If fewer than 6 items on the row, the items expand to fill the space. 
    • Use exactly 5 linked tiles with the View All option.
    • Use 7-12 total titles across 2 rows. If fewer than 12 tiles, empty slots are left blank. 
  • If you want an introductory heading or text, use a Section Header above this component.

Image Guidelines

Format

  • SVG and PNG supported. 
  • Transparency strongly preferred for all logos. Transparency required if linking the tiles (since any background will show up on hover and look unprofessional). 
  • Size constraints: 160px X 120px (i.e. 4:3). 

Content

  • Single monochrome logo per image.
  • No co-branding or color.
