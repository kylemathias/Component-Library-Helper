Component name: Prose                         Nav Bar (Optional- Include or Don’t Include):

Guidelines:

Headline (required)- Character limit 30

Subhead (optional)- Character limit is 30

Body (required)- Character limit is 200; Can have linked text but not suggested.

CTA (optional)- Choose one of the following:

CTA Link

    • 0-2 links, inline.
    • Character limit is 45. Keep it as short as possible. 

CTA Button

    • 0-2 Buttons, inline.
    • Character limit is 25.
    • If two buttons, character limit is 10 per button.

Use When

  • Composing long form text.
  • Alongside other components like Title, Callout, and Section Header.

Don’t Use When

  • Composing blog posts.
  • Composing customer stories.

Editorial

Body

  • No character limit.
  • Allowed: Headings, paragraphs, inline links, bullets, ordered lists, horizontal rules, and images. 
  • Alt text required for images. Images display at native size, so size your images appropriately. 

Sidebar (optional)

  • Nav option: When enabled, a nav bar is automatically generated from the RTE's headings. Anchor links are added automatically. 
  • RTE option: When enabled, a second RTE area appears in the left/previous sidebar. 

CTA (optional)

  • 0-2 CTAs in a structured spot at the end of the component.
  • Character limit is 25.
