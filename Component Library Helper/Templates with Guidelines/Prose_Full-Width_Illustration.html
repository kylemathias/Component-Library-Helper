Displays a large illustration to add content or visual interest to a prose article.

Prose Full-Width Illustration is part of a collection of components which must be used inside a ProseGroup Region, which provides layout via a special grid.

Use When

  • Punctuating an article with a large, powerful image. 
  • Breaking a long article into discrete visual blocks (usually with a brand 3d render or a brand stock photo). 

Don't use when

  • Illustrating a technical story with a diagram or video. Use the Prose Inline Media Player. This component isn't well-suited for technical diagrams.

Editorial

  • Image (required)
    • Intended for wide landscape images, especially rendered brand patterns or brand stock photos. If the page uses the Table of Contents component set to be sticky, then the left side of this image will be covered by it. In that case, ensure the POI is on the right of the image. 
    • Optionally, use a key frame image and provide the BrightCove videoID to open a video in the Brightcove modal. Embedded players and click-to-enlarge images are not supported - see the Prose Inline Media Player instead. 
  • Caption (optional)
    • Plain text. 
    • 500 character limit. Shorter is better.
