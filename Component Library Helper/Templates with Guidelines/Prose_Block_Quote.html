A high-visibility block quote/pull quote.

Prose Block Quote is part of a collection of components which must be used inside a ProseGroup Region, which provides layout via a special grid.  

Template Copy Block

 

Component name: Prose Block Quote

Guidelines:

Body (required)- Character limit 200.

Attribution (optional)- Character limit is 60.

CTA (optional)- Choose one of the following:

CTA Link

    • 0-1 links, inline.
    • Character limit is 45. Keep it as short as possible. 

CTA Button

    • 0-2 Buttons, inline.
    • Character limit is 25.
    • If two buttons, character limit is 10 per button.

Use When

  • Displaying a pull quote on an article page inside a Prose Container (pull quotes come from the article's text). 
  • Displaying a block quote on an article page inside a Prose Container (block quotes transcribe something a real person said). 
  • Displaying a key takeaway or call out on an article page inside a Prose Container . 

Don't use when

  • Displaying a quote outside the ProseGroup Region, such as on T2 and T3 marketing pages. Instead, use Quote Band. 

Editorial

Body (required)

  • Text field.
  • Character limit is 200.

Attribution (optional)

  • Text field.
  • Character limit is 60.

CTA (optional)

Content recommendation: Only use this link after a quote to direct users to a success story or similar content by the author. 

Choose one of the following (see CTA Patterns for usage):

CTA Link (Default)

    • Text link only.
    • Character limit is 30.

CTA Button

    • 0-2 Buttons.
    • If two buttons, character limit is 10 per button.
    • Character limit is 25.
