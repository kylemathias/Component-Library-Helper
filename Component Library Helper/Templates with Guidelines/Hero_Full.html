Use When

  • Signaling the importance of top-level marketing pages by placing it directly below the site header.

Don’t Use When

  • Introducing minor or serial pages, such as articles, blog posts, press releases. Consider the Title component instead. 
  • Introducing mid-level marketing pages. Use the Hero Side x Side instead.
  • Placing it anywhere but directly below the site header.

 

Editorial

Breadcrumb (required)

  • This element is inserted automatically and is based on site hierarchy. It cannot be removed by author.

 

Pill (optional)

  • Link to the relevant solution hub (only) using this distinctive eyebrow 'tag'.

 

Headline (required)

  • Text field.
  • Character limit 45.
  • Phrase the headline as a traditional page title, rather than a catchy tagline. For products, just name the product.  

 

Description (required)

  • Text field. (Note: dev is potentially converting this to an RTE field.)
  • Character limit is 200.
  • Support the title with more description. 

 

Image (required)

  • Image is never clickable.
  • Right-aligned. Not flippable.
  • See Visual Style for image composition guidelines.

 

CTA Button (optional)

  • 0-2 CTAs
  • If no CTAs, use the same character count for headline and description.
  • Character limit is 30.
  • Recommended copy:
    • "Get in touch" (sales contact form)
    • "Read the paper" (link to PDF)
    • "Watch the video" (modal with player)
  • See CTA Patterns for usage.
