Alt text:  

 

 

 

 

 

Use When

  • Grouping a set of peer content snippets or navigation links that go to related pages.

Don’t Use When

  • Displaying parallel alternatives or elements. Use Tabbed Band instead.
  • Displaying a collection of unrelated links. Use Offset Cards instead.
  • Required visual impact is minimal. Use Feature Tiles instead.
  • Subdividing a set of peer navigation links with a common introduction. Use Tabbed Band Tiles. 

 

Editorial

Per Tile Headline (required)

  • Text field.
  • Character limit is 30.

Per Tile Body  (required)

  • RTE field. Superscript and bold okay. No other formatting, linking, or bullets allowed. Links will break this component.
  • Character limit is 60.

Per Tile CTA (optional)

  • Text link only. Label is optional. If no label provided an arrow will still appear.
  • Character limit is 15.
  • Entire tile is interactive. 

 

Layout

  • # of tiles: 2-8 tiles
  • Tiles will stack vertically downward in a 2-by-N column.
  • When linked, the entire tile is clickable.
