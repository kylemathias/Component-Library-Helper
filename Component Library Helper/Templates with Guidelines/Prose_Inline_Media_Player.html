Displays embedded media in a prose article.

Prose Inline Media Player is part of a collection of components which must be used inside a ProseGroup Region, which provides layout via a special grid.

Use When

  • Illustrating article content with an image. 
  • Illustrating a technical point with a diagram, video, or 3D render. 
  • Supporting article content with a media file, such as a video. 

See requirements for current status of various embed methods. 

Editorial

Media object (required, select from:)

  • Image, optionally linked to a resource. 
    • Link a PDF, which opens in a new tab.
    • Link a high-resolution version of the image (click to enlarge), which opens in a modal component. Two images are required:
      • Inline image: 926px by 512px (=16:9. You can vary the height if needed, to support a 4:3 image). 
      • A higher resolution version. Appropriate size depends on content, target >1800px wide. 
  • Image key frame of a video that opens in a modal player.
    • Note: this option can include audio files, as the Brightcove player can handle these. 

Caption (optional)

  • Plain text. 
  • 500 character limit. Shorter is better. 
  • Collapses if absent (default).

Visual Style

Placement

  • Use in the main content region inside a ProseGroup Region.
  • Can be used multiple times on a page. 

Layout
