Use When

  • Displaying multiple calls to action of equal weight.
  • Displaying snippets of microcopy with equal weight. 
  • When displaying accolades, such as Gartner and Forrester reports.

Don’t Use When

  • If you have an indeterminate number of different destinations. Consider Feature Tiles or Offset Cards instead.
  • High visual impact is needed. Use Card Band Images instead.

Editorial

  • Text field.
  • Character limit is 30.

Per Card Body  (required)

  • Character limit is 60.
  • Rich text field. Bullets, inline links, and paragraphs allowed. 

CTA Link per card (optional)

  • 0-3 text links
  • Text field.
  • Character limit is 15.
