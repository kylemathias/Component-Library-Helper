Prose displays long form narrative text.

Prose is part of a collection of components which must be used inside a ProseGroup Region, which provides layout via a special grid.

CTA (optional)- Choose one of the following:

CTA Link

    • Character limit is 45. Keep it as short as possible. 

CTA Button

    • 0-2 Buttons, inline.
    • Character limit is 25.
    • If two buttons, character limit is 10 per button.

Editorial

Elements per section

Pill, optional above a major heading.

Heading (optional)

  • The component has a major heading. Any subcomponents (subsections) have a minor heading, which is smaller. Subsection body RTE can contain one or more small headings.) See the "Organization of Sections" below for more guidance. 

Body (required)

  • Rich text. Bullets, lists, links, and character styling allowed. No character limit. 
  • May contain small headings. 
  • May contain an image, although we recommend authors switch to the full-width illustration or inline media components instead.
    • Alt text required for images. Images display at native size, so size your images appropriately. 

CTA (optional)

  • Choose one of the following (see CTA Patterns for usage):

  • CTA Link
    • 0-2 links, inline. 1 link preferred.
    • Character limit is 45. Keep it as short as possible.

  • CTA Button

    • 0-2 Buttons, inline.
    • Character limit is 25.  

Organization of sections

Prose is divided into sections and subsections, following an article's heading hierarchy.
