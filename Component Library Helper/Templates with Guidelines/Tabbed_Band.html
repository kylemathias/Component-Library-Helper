Use When

  • Displaying 2-5 examples grouped by a unifying topic.
  • More examples need to be displayed than will fit in a single band.
  • One of the examples can serve as a compelling summary of the set of examples.

 

Don’t Use When

  • Displaying multiple unrelated examples. 
  • Displaying only one example.
  • Telling a sequential story.
  • Displaying customer quotes. Use Quote Band instead.
