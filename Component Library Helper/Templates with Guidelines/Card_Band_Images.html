Editorial

Per Card Headline (required)

  • Text field.
  • Character limit is 30.

Per Card Body (optional)

  • Character limit is 60.
  • Rich text field. Bold okay. Additional formatting and links discouraged. 

Per Card CTA Link (optional)

  • 0-2 text links
  • Text field.
  • Character limit is 15.
  • Label is optional. 

Per Card Image (required)

  • Image is never clickable.
  • Do not mix logos with standard images. Use all logos, or all images.
