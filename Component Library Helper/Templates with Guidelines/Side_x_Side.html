Headline (required)

  • Character limit is 60.
  • Must be key term relevant. 
  • If used as a hero, set the headline as an H1-level heading.

Body  (required)

  • Character limit is 200.
  • Can have linked text but not suggested.
  • Free-form copy area can include: text, bulleted list, or a numbered list. 

CTA (optional)

Choose one of the following (see CTA Patterns for usage):

CTA Link

    • 0-2 links, inline. 1 link preferred.
    • Character limit is 45. Keep it as short as possible. 

CTA Button

    • 0-2 Buttons, inline.
    • Character limit is 25. If two buttons, character limit is 10 per button.

Link List

    • 2-3 Links.
    • Character limit is 30 per link.
    • If only one link, switch to using CTA Link.

Image (required)

  • Optionally, link a video thumbnail to open that video in a modal layer. This is the only option for linking the image. The component provides the play button, which should not be embedded in the thumbnail image. 
  • If the image is considered content, the title and alt text must be key term relevant.
