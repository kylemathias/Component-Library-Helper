Component name: Accordions

Headline (required)- Character limit is 60

Sub header (optional)- Character limit is 90

Panel Content (required, use 1 or more times)

  • Rich text field. Bullets, inline links, and paragraphs allowed. 
  • H2, H3, H4 headings allowed
  • Images Allowed
    • Alt text required for images. Images display at native size, so size your images appropriately. 
  • Horizontal Rules Allowed

Panel CTA (optional, use 0 or more times). 

  • Text link only. Label required. Character limit is 30.
  • Insert the CTA as a separate content block within an accordion panel.
