Template Block

 

You can copy and paste the table into your template directly from netapp.com page, it should format correctly and you can then have a SME edit the table within the NewsCred content editor

 

 

Use When

  • Helping to compare features for a range of similar products.

Do not Use When

  • Presenting a general overview of all products. Consider the Showcase Product. 
  • Navigation to several products. Consider Image with Tiles or Tabbed Band Tiles. 
  • Displaying a simple table. Consider RTE - Table.
